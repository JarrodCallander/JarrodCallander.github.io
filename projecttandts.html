Tools and Technologies

The tools and technologies required to build this app include an app design platform such as BuildFire or Xamarin to design the UI and functionality of the app. Because the app would store personal information server side, the Australian Privacy Principles (APPs), Telecommunications Act, Anti-Money Laundering and Counter-Terrorism Financing Act will apply (International Comparative Legal Guides International Business Reports, n.d.) and therefore stringent security measures would need to be used in the storage of the data. Cloud storage would be the solution for storing the data, however, to maintain security the storage would need to be hosted in Australia. Encryption such as Elliptic-curve Diffie-Hellman (ECDH) or Elliptic Curve Digital Signature Algorithm (ECDSA) would be used to encrypt financial transactions (Guidelines for using cryptography, n.d.).
